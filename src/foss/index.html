FOSS@BPHC Induction

Inspired by the Foss@Amrita, we have begun a BPHC version of FOSS andd we are now inducting members. Unlike most other cl ubs, this club needs more than a little commitment from you and to test if you have what it takes, the induction challenge has been described below.

Goal

Create a program/script in your favorite programming language to perform the following tasks:

  1. Parse a youtube "playlist" page to extract links of all videos in the playlist into a file.
  2. One by one download all the videos to a folder.

Rules

  1. Should be a command line program.
  2. Cross platform compatibility is not required.
  3. At first run, provide option for setting the downloads folder.
  4. The code should be under 150 lines.

Tips

You can use any programming language you want, but as an example, I'll list out some of them here:

  1. Python
  2. JavaScript
  3. Bash Scripting
  4. C++
  5. Java

With that out of the way, you are also free to utilize any modules or plugins available for your language/OS of choice. If you do so, you should also provide a small installation script to handle that installation. An example of a small installation script could be:


#! /usr/bin/env bash
# install a package you need for your project.
sudo apt-get install my-dependency -y
# Supposing you used nodejs, let's install all your node dependencies with this command.
sudo npm install
echo "Installation successful. You may now run the project."

Marking Scheme

Out of 100 marks, the division will be as follows:

  • 60 marks for a working program.
  • 10 marks for well written code. (covers indentation, descriptive variable names, comments, etc.)
  • 10 marks for providing the ability to continue downloads if once interupted.
  • 10 marks for a good installation script/instructions. If you have a stand alone program, you by default get 10.
  • 10 marks, allgiven by default, with 1 mark cut for each 10 lines of code above the 150 lines limit.

Submission

Submit your assigments by either:

  1. Creating a Github/Gitlab repository and sending the link. (preferred)
  2. Creating a zip file with all the code and sending the file.

Email the assignments to aero31aero@gmail.com before 10 PM on 5th November, 2016. We will interview the top 10 people on the basis of the rank list. Good luck!
